httpRange

Posted on 2017-02-16 | In Objective-C , Other , httpRange

设置请求头Range可以指定每次下载数据包大小

bytes = 0 - 499 从 0 - 499的头500个字节

bytes = 500 - 999 从 500 - 999的第二个500字节

bytes = 500- 从500字节以后的所有字节

bytes = -500 最后500个字节

bytes = 500 - 599 800 - 899 同时指定几个范围

  • 用于分隔

前面的数字表示起始字节
后面的数字表示截至字节,没有表示到末尾
，用于分组
