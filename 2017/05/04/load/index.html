load

1. load 是什么?

  • 定义在NSObject.h头文件中
  • load方法在这个文件被程序装载时调用, 只要是在Compile Sources中出现的文件总是会被装载, 无论这个类是否被用到;
  • load方法在main函数之前调用;

2. load 怎么调用?

  • 如果在一个类中实现 load 方法, 则会先调用其父类的 load 方法, 是系统自动调用的;
  • 如果一个类没有实现 load 方法, 则系统不会调用它的父类;
  • 类别中调用 load 方法, 遵循上面两条;
  • load 方法不能手动调用;
  • load 是线程安全的, 内部使用了锁, 应该避免在 load 中阻塞线程
